# TOSM Damage Calculator

Public Tree of Savior Mobile damage calculator built with Next.js.

## Run locally

```bash
npm install
npm run dev
```

## Quality checks

```bash
npm run test
npm run lint
npm run build
```

## Deploy on Vercel

1. Push this project to a Git repository.
2. Import the repository in Vercel.
3. Keep defaults (Framework preset: Next.js).
4. Deploy and share your public URL.
